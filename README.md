# User Bio by David Corlette

![License](https://img.shields.io/badge/license-MIT-blue.svg) 

A [Flarum](http://flarum.org) extension. Add a user's student and feeder school info to their profile.

### Installation

Install manually with composer:

```sh
composer require dgc/user-students:"*"
```

### Updating

```sh
composer update dgc/user-students
```
